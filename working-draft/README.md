# Avanti proposal static working draft

This folder holds the live static HTML working draft for the Avanti proposal inside the proposal-ops structure.

Location in proposal-ops format:
- `docs/leads/AVT-avanti-restaurant/working-draft/` — live proposal draft

Files to edit in visual editors:
- `index.html` — English proposal page
- `bangla.html` — Bangla proposal page
- `styles.css` — all page styling
- `script.js` — tiny behavior layer (print button + initial reveal)
- `assets/images/` — local image assets used by the proposal
- `assets/review/` — saved review screenshots

Open `index.html` directly in a browser or serve this folder with any static server.
